Java Class

Class란?
-객체 지향 프로그래밍에서, 데이터와 그 조작 절차인 메소드를 정리한 객체의 추형을 정의 한 것. 이를 통하여 동종의 객체를 정리하여 다룰수 있게된다.
(feat.국어사전)
쉽게말해 데이터와 메서드를 정리한 객체를 정의한 것,
객체를 정리하고 다룬다.

객체
객체 지향 프로그래밍이나 설계에서 테이터와 그 데이터에 관련되는 동작(절차, 방법,기능)을 모두 포함한 개념
예)
기차역에서 승차권을 발매하는 경우 실체인 손님과 동작인 승차권주문은 하나의 객체이다
실체인 역무원과 동작인 승차권발매도 하나의 객체이다.
데이터(실체)와 그데이터에 관련되는 동작(절차, 방법,기능)을 기억하면 된다.


-객체는 사용할수 있는 실체를 의미하고 클래스는 객체를 만들기 위한 설계도와 같다.

-자바에서는 이러한 설계도인 클래스를 가지고, 여러 객체를 생성하여 사용할수있다.
-클래스는 다양한 자료형의 데이터를 묶어 관리 할수 있고 데이터를 처리하는 다양한 기능까지 함께 관리하는 문법요소이다.


클래스의 생성자로 객체를 만드는 과정을 '인스턴트화'라고 하고 인스턴스화로 만들어진 객체를 '인스터스'라고 한다.
클래스는 바로 사용할수 없고 반드시 객체를 생성해 객체 안에 있는 필드, 메서드 및 이너클래스를 사용해야 한다.
예시)붕어빵
붕어빵을 만들려면 틀이 필요함
붕어빵틀이  Class
만들어진 붕어빵이 객체를 의미함

클래스로부터 객체를 생성하는 방법과 객체를 활용하는 방법
↓                       ↓                    ↓
붕어빵 기계에서 붕어빵을 찍는 방법과 붕어빵을 맛있게 먹는 방법


클래스 외부구성요소
-패키지 비슷한 목적으로 생성된 클래스 파일들을 한곳에 모아둔 폴더를 의미
-임포트
-외부클래스

클래스 내부구성요소
-필드
-메서드 클래스의 기능에 해당하는 요소 
예를 들어 사람 클래스라면 먹기,잠자기,공부하기
          자동차 클래스라면 전진하기, 후진하기, 회전하기 가 메소드로 구성된다.
-생성자 객체를 생성하는 역할을 지닌 클래스의 내부구성요소
특징 1. 반드시 클래스명과 동일한 이름으로 지어야 함.
     2. 리턴타입이 없다.
-이너클래스
